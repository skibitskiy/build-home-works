# Задание

Для данного домашенго задания предусмотрен тест на платформе геткурс. Для его успешного выполнения внимательно изучите материалы лекций, а так же дополнительно разделы документации у сборщиков:

- [webpack hooks](https://webpack.js.org/api/compilation-hooks/)
- [webpack source code](https://github.com/webpack/webpack)
- [rollup hooks](https://rollupjs.org/plugin-development/#build-hooks)
- [rollup source code](https://github.com/rollup/rollup)
- [esbuild hooks](https://esbuild.github.io/plugins/)
- [esbuild source code](https://github.com/evanw/esbuild)
- [vite hooks](https://vite.dev/guide/api-plugin.html)
- [vite source code](https://github.com/vitejs/vite)
